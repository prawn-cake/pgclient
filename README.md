pgclient
=============

pgclient - yet another pool-based psycopg2 wrapper. 


Quick start
===========




Installation
============
System dependencies
-------------------

Install the following dependencies via `.deb` or `.rpm` packages

* python-dev
* libpq-dev

Install package
---------------

    pip install pgclient
